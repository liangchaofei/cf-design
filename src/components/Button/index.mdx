## API文档
|  参数  |  说明  |  类型  |  默认值  |
|  ---   |  ---  |  ---  |  ---  |
|  className  |  自定义类名  |  string  |  |
|  type  |  按钮类型 primary,warning, info, default, pure  |  string  |  default  |
|  shape  |  按钮形状circle ,radius(默认)  |  string  |  radius  |
|  block  |  是否是块状  |  boolean  |  false  |
|  onClick  |  按钮点击事件  |  (event) => void  |  |
