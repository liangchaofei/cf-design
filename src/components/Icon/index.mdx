## 用法
``` js
import { Icon } from 'cf-design'

// 使用
<Icon type="FaBeer" />
<Icon type="FaAccessibleIcon" size="24px" />
<Icon type="FaAlipay" size="24px" style={{color: 'red'}} />
```


## API文档
|  参数  |  说明  |  类型  |  默认值  |
|  ---   |  ---  |  ---  |  ---  |
|  type  |  icon类型  |  string  |  |
|  size  |  icon大小(px，lg，xs)  |  string  |  |
|  rotation  |  旋转角度  |  number  |  |
|  style  |  图标样式  |  object  |  |
